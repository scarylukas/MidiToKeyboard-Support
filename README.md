Privacy Policy – ScaryTeam Software
Midi To Keyboard
Effective date: 2025-12-30
1. Introduction
This Privacy Policy explains how the Midi To Keyboard application, developed by ScaryTeam (“we”, “us”), processes user information.
We respect user privacy and designed this application to operate without collecting or transmitting personal data, in accordance with applicable data protection laws, including the GDPR.
2. Scope
This policy applies specifically to the Midi To Keyboard application distributed by ScaryTeam, including versions published on the Apple App Store.
The application does not operate any proprietary backend servers and does not require user accounts.
3. Data We Process
a) Local data
The application may store user-defined configuration settings locally on the user’s device, such as:
MIDI device mappings
Keyboard mapping preferences
Application configuration options
This data:
is stored only locally on the user’s computer,
is not transmitted to ScaryTeam or third parties.
b) Cloud synchronization
The application does not use cloud synchronization and does not store data in iCloud or other cloud services.
c) Platform services
The application does not integrate Game Center, analytics, advertising SDKs, or third-party services.
No platform-provided personal identifiers are accessed or processed.
4. Purpose of Processing
Local data is processed solely to:
enable proper operation of the application,
store user preferences and mappings.
5. Legal Basis (GDPR)
Processing is based on:
Art. 6(1)(b) – performance of a license agreement (providing application functionality).
No consent-based or legitimate-interest processing of personal data is performed.
6. Data Storage and Retention
Local data is stored on the user’s device until:
deleted by the user, or
the application is removed.
ScaryTeam does not maintain any external databases containing user data.
7. Data Sharing
No data is sold, shared, or transferred to third parties.
No marketing, tracking, profiling, or analytics services are used.
8. Security
Local data is protected by operating system sandboxing and security mechanisms.
No network transmission of user data occurs.
9. User Rights (GDPR)
Where applicable, users have the right to:
access, rectify, or delete their data,
restrict or object to processing.
Because all data is stored locally, users can exercise these rights by deleting the application or its local data.
10. Children
The application does not knowingly collect personal data from children.
11. Changes to This Policy
This policy may be updated to reflect legal or technical changes.
The current version will always include an updated effective date.
